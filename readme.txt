Create Custom Columns for your Post Types easily

== Description ==

Amazing Improvements in Version 1.0! Check it out!
Tested with 3.8.1 and higher only, but could work with previous versions.

Add more informations to your lists with Custom Columns.

= Configuration =

1. Click on the appeared menu link under "Settings".
2. Choose one of the posttypes.
3. Use the + to add a new column.
4. Double click on the tab to change the column name.
5. In the active tab, click x to remove the tab.
6. Click on "Options", so you can make the column sortable. Then just choose a type and a value.
7. On the elements header click + to add a new element.
8. For each element you can choose a type and define several options.


== Installation ==

= WordPress Backend =
1. Just search for "custom columns" and click install. 

= Manually =
1. Upload `custom-columns`-Dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your Custom Columns!
